# 地图 GeoJSON（离线）

本目录用于**离线展示**中国地图与省份地图，不访问在线数据。

- **中国全图**：`100000_full.json`，用于首页「全国监测分布图」
- **省份地图**：`{adcode}.json`，例如 `110000.json`（北京市）、`310000.json`（上海市）

**生成方式**：在项目根目录执行一次下载脚本（需联网）即可拉取到本地：

```bash
cd frontend && node scripts/download-geo.cjs
```

之后前端仅从 `/geo/` 下读取上述文件，无需再访问外网。

常用 adcode：110000 北京，120000 天津，130000 河北，310000 上海，320000 江苏，330000 浙江，370000 山东，440000 广东，500000 重庆，510000 四川 等。
